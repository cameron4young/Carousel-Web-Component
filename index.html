Description

Custom Carousel (<custom-carousel>) is an open source Web Component that can be used to create a Carousel display on webpages, without any knowledge of JavaScript.

To see how the component works, check out the demos above! Check out the Custom Carousel GitHub repository to learn more about how it works and use it for yourself!
